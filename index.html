Docs Comment Exporter

Export Google Docs comments to Markdown, CSV, or JSON — instantly.

  • View on Chrome Web Store

What It Does

Docs Comment Exporter is a lightweight Chrome extension that lets you export all comments and threads from a Google Doc in one click.

  • Instant comment detection & preview
  • One-click export — no setup required
  • Export to Markdown, CSV, or JSON
  • Includes full metadata (authors, timestamps, threads)

Privacy

Docs Comment Exporter runs entirely in your browser. Your documents and comments are never sent, stored, or processed externally.
