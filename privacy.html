Privacy Policy

Data Access and Processing

Docs Comment Exporter accesses Google Docs comments only within the user’s browser for the purpose of exporting them to user-selected formats (Markdown, CSV, or JSON).

  • All data processing occurs locally in the browser
  • No data is transmitted to external servers
  • No data is stored permanently
  • No data is shared with third parties

The extension does not collect analytics, track users, or use cookies. Access to Google Docs content is limited strictly to the active document opened by the user and only occurs when the user explicitly initiates an export action.

Data Retention

Docs Comment Exporter does not retain any user data. Exported files are generated locally and saved only at the user’s discretion.

Third-Party Services

This extension does not use third-party services, APIs, or analytics tools.
